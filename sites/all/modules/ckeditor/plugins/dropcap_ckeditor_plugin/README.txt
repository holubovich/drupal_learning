Improve the content authoring experience by using CKEditor's new widget 
interface.

-- Requirements --
* [CKEditor Drupal module](http://drupal.org/project/ckeditor)


-- SUMMARY --

This module add dropcap functionality in ckeditor.

-- REQUIREMENTS --
  No

-- INSTALLATION and configuration --
* Install the module at `/admin/modules`
* GO TO 
(for example `admin/config/content/ckeditor/edit/profile-name`):
    * Open the Editor Appearance fieldset. Towards the bottom under Plugins, 
check the box next to: Plugin to add dropcap widget.
  * Drag the dropcap icon to current toolbar.
